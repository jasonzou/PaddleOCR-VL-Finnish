# PaddleOCR-VL-1.5 Finnish

Fine-tuning [PaddleOCR-VL-1.5](https://huggingface.co/PaddlePaddle/PaddleOCR-VL-1.5) for Finnish OCR using LoRA SFT, built for the Paddle Global Model Derivative Competition.

---

## Setup

- CUDA 12+, VRAM 24G+ with conda installed 

```bash
conda create -n ppocr python=3.11 -y
conda activate ppocr
pip install -e '.[paddlefleet]' --extra-index-url https://www.paddlepaddle.org.cn/packages/nightly/cu129/ --extra-index-url https://www.paddlepaddle.org.cn/packages/stable/cu129/
git clone https://github.com/jasonzou/PaddleOCR-VL-Finnish
cd PaddleOCR-VL-Finnish
pip install datasets pillow pyarrow tqdm sickle pdfplumber pymupdf requests python-Levenshtein
```

## Datasets

| Name | Source | Rows | Script |
|------|--------|-----:|--------|
| **hf** | HuggingFace: caveman273/aida-* | ~57k | `build_ocr_dataset.py hf` |
| **digi-natlib** | Github: sdrobac/nodalida2017 | ~65k | `build_ocr_dataset.py digi-natlib` |

All datasets stored as `data/<name>/train.parquet` with embedded PNG bytes.
Schema: `source`, `file_name`, `text`, `image`.

Generate JSONL splits (90/5/5):

```bash
python build_ocr_dataset.py hf --gen
python build_ocr_dataset.py digi-natlib --gen
```

## Scripts

### build_ocr_dataset.py
Build datasets from various sources.

```bash
python build_ocr_dataset.py hf --gen                    # HuggingFace datasets
python build_ocr_dataset.py digi-natlib --gen           # Nodalida2017
```

### check_dataset.py
Explore or export parquet datasets.

```bash
python check_dataset.py                          # Display sample images
python check_dataset.py --summary                # Dataset statistics
python check_dataset.py --save                   # Save images to samples/
python check_dataset.py --n 20 --output-dir extracts
```

### eval.py
Run batch inference on JSONL test data.

```bash
python eval.py \
  --model_name_or_path ./models/your_model \
  --data_path data/digi-natlib-gen/test.jsonl \
  --data_dir ./data/digi-natlib-gen/ \
  --output_path results.jsonl \
```



### calculate_ned.py
Calculate Normalized Edit Distance (NED), Character Error Rate (CER), and Word Error Rate (WER) from model output JSONL files generated by eval.py.

```bash
python calculate_ned.py output/results.jsonl
```

### test_single_image.py
Test single image inference.

```bash
python test_single_image.py \
  --model_name_or_path ./model \
  --image_path ./image.jpg \
  --question "OCR:"

# Or use predefined tasks
python test_single_image.py --model_name_or_path ./model --image_path ./img.jpg --task region
```

|-------|------|-------|
| **hf** | HuggingFace | `trainv1-hf.sh` / `lorav1-hf.sh` |
| **digi-natlib** | digi + natlib | `trainv1-dn.sh` / `lorav1-dn.sh` |

Export and merge LoRA:

```bash
bash merge-lora.sh
```

## Project structure

```
build_ocr_dataset.py # Dataset builder
check_dataset.py     # Dataset explorer
eval.py              # Batch inference
calculate_ned.py     # Calculate NED, CER, WER
test_single_image.py # Single image test
data/                # Datasets
  hf/                # HuggingFace parquet + JSONL
  digi-natlib/       # Nodalida parquet
  digi-natlib-gen/   # JSONL splits + images
output/              # Inference results
models/              # Model checkpoints (not tracked)
```
